Welcome to Change-HI

You have reached the home page for the curriculum modules associated with the University of Hawaii EPSCOR Change-HI Program. These modules are designed to support the various programs associated with the Hawaii Data Science Institute, including CI-TRACS, CITRUS, and the Data Fellows program.

Some modules are designed to be used as curriculum for one day workshops, while others can be used asynchronously (i.e. the student can work through the material at their own pace and time).

About this site

Modules have one or more of the following components:

  • Prerequisites, describing skills you should acquire in order to be ready for the module.
  • Learning outcomes, describing the major goals for the module.
  • Readings and other online resources, providing background or reference material.
  • Experiential learning activities, providing a structured set of exercises enabling you to acquire mastery of material.
  • Assessments, which help you determine if you have acquired mastery of the material and to provide feedback about the module.
